# CaptureColour
With this Add-in all of your saved colours are available to all workbooks! Just open the Add-in, select your saved colour and click ‘+ to recent colours’.

Installing Capture Colour for Excel

1)  Unzip the Capture Colour zip file to extract the contents
2)  Move the add-in and the CaptureColourLog.txt to your local Microsoft AddIns directory.
3)  Open Excel
4)  Click the File tab
5)  Click Options
6)  Click the Add-Ins category on the left bar of the Excel Options Dialog Box
7)  In the Manage box near the bottom, click Excel Add-ins, and then click Go. The Add-Ins dialog box appears.
8)  Click Browse
9)  Navigate to the ColourCapture add-in and select it.
10) Make sure the checkbox next to "Capture Colour" is checked, then click OK.
11) If the add-in keeps disappearing when you restart Excel, follow the directions below.


If the add-in keeps disappearing when you restart Excel, you might need to unblock it.      
In conjuction with the above instructions please refer to the included pdf file:
  Installing an Excel Add-in - Security Update.pdf
This step by step guide created by ExcelCampus.com will guide you through this process.
